# CreditWise

## Overview

CreditWise is a machine learning project focused on analyzing customer financial information to predict creditworthiness and assess credit risk. The project demonstrates the complete workflow of a data science project, including data preprocessing, exploratory data analysis, feature engineering, model training, and performance evaluation using Python and Jupyter Notebook.

## Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature engineering
* Machine learning model training
* Model evaluation using standard classification metrics
* Prediction of customer credit risk

## Technologies Used
* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Dataset
The dataset is not included in this repository due to licensing and repository size considerations. Please download the dataset separately and place it in your local project directory before running the notebook.

## Installation
Clone the repository:
git clone https://github.com/Disha-Lahoti/CreditWise.git

Navigate to the project folder:
cd CreditWise


Install the required dependencies:
pip install pandas numpy matplotlib scikit-learn jupyter

Launch Jupyter Notebook:
jupyter notebook

Open credit_wise.ipynb and run the notebook cells.
